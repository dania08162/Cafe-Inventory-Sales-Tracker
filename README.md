# Cafe-Inventory-Sales-Tracker
One of my final projects in my Python class!

#Purpose
I built this project to practice programming concepts and to simulate how real businesses track inventory and sales. It helped me understand how data is stored, updated, and processed in real-world systems.

#Features
- Loading inventory and sales data from CSV files
- Processing sales and updating stock quantities
- Calculating total revenue automatically
- Identifying low-stock items for restocking

#Instructions
1. Run the setup cell to generate the sample CSV files.
2. Copy and paste the code into any Python environment (I used Google Colab).
3. Run the program to process the sales data and view the total revenue and low-stock items!

#Technologies Used
- Python
- Google Colab
- CSV file handling

#What I Learned
I learned how to work with CSV files using csv.DictReader, how to store and manipulate data using lists of dictionaries, and how to process real world data using loops and functions. I also improved my understanding of data conversion (strings to integers/floats), updating values dynamically, and organizing code into clear, reusable functions.
